# Электронная таблица на C++

Проект реализует функциональность электронной таблицы с поддержкой формул, обработкой зависимостей между ячейками и кэшированием результатов.

## Особенности

✅ Поддержка текстовых и числовых значений  
✅ Вычисление формул с базовыми арифметическими операциями  
✅ Автоматическое обновление зависимых ячеек  
✅ Обнаружение циклических зависимостей  
✅ Оптимизация производительности через кэширование  
✅ Чекер корректности позиций ячеек
